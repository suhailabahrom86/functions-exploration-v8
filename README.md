# Functions Exploration V8 Professional — Teacher Access Code

Teacher Monitor and Clear Leaderboard are now protected by an access code.

Default teacher access code:

`SB-V8-2026`

Security behaviour:
- The code is verified using a SHA-256 hash.
- Access remains unlocked for 120 minutes in the current browser tab.
- After 3 incorrect attempts, access is locked for 30 seconds.
- Teacher Monitor, Refresh, Export CSV and Close continue to work after unlocking.

Important:
This is client-side access protection suitable for classroom use. For strong security, use Firebase Authentication with approved lecturer email accounts.
